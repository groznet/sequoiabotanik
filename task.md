Create a responsive web page template using **HTML, Tailwind CSS, and vanilla JavaScript (only where necessary)**.

### Design, Look and Feel Rules

Design and overall look and feel rules:
design a highly creative, nature-inspired website header for a premium landscaping brand “Sequoya.” Avoid traditional straight horizontal layouts. Instead, create a fluid, organic composition with curved, layered shapes that feel like natural terrain, hills, or forest depth.

The header should feel immersive, as if multiple layers of nature overlap: foreground elements (leaves, branches, soft shapes), midground content (text and CTA), and a subtle background (landscape, trees, sky gradients). Elements should partially overlap and flow into each other, creating depth and intrigue.

Use smooth, asymmetric curves instead of rigid lines. Sections should blend into one another like waves or land contours. Introduce soft masks, blurred edges, and layered silhouettes inspired by nature.

Apply a refined green palette (deep forest, moss, olive) with soft natural light gradients. Add subtle motion or parallax feel where background and foreground move at different speeds.

Typography remains clean and readable, but integrated into the environment—not boxed. CTA elements should feel embedded into the layout, not floating artificially.

### Requirements:

* The layout must include:

  * Header
  * Sidebar
  * Main content area
  * Footer

* Each section must be **modular and reusable**, stored as separate HTML files inside a `/components` directory.

* Use **JavaScript to dynamically load (include) these components** into the main layout.

* Implement a **mobile-friendly design**:

  * Sidebar should be hidden by default on smaller screens
  * Include a **menu toggle button (hamburger icon)** in the header
  * Clicking the button should show/hide the sidebar

### Constraints:

* Keep the structure clean and minimal (focus on functionality, not styling)
* Use Tailwind only for layout and responsiveness
* Ensure the template works immediately when opened in a browser (no build tools)

### Output:

* Provide a **fully working base template**
* Include:

  * `index.html`
  * `/components/header.html`
  * `/components/sidebar.html`
  * `/components/footer.html`
  * Any required JavaScript
